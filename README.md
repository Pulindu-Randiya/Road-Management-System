Roadsafe Urban Hazard Detection System

This is an academic group project focused on designing a smart road hazard detection and mapping system for urban areas.

The project aimed to identify and report road hazards such as potholes, puddles, and obstacles using modern technologies including drones, LiDAR sensors, fixed cameras, traffic analytics, and citizen reports through a mobile application.

My Contribution
My role focused on hardware and drone operations, including planning the use of drones, LiDAR sensors, cameras, and supporting technology for hazard detection. I also contributed to project planning, stakeholder analysis, risk management, and technology stack documentation.
